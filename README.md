## 为什么要进行翻译工作（Why to do）？

- 利用好零碎化时间
- 优质资源的引入
- 翻译的过程就是学习的过程，也是知识总结和沉淀的过程

## 翻译工作有什么好处（What are the benefits）？

- 有一定的成就感和获得感
- 结交到和自己兴趣相同的伙伴，一同进步
- 知识的积累与沉淀，对自己有一个正向反馈的激励作用
